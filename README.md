
# To-Do Application

A streamlined task management experience engineered for clarity, speed, and focus. This web-based To-Do app transforms scattered thoughts into structured execution using pure HTML, CSS, and JavaScript.

# Core Capabilities
Add, edit, and delete tasks effortlessly
Mark tasks as completed with visual feedback
Persistent task storage using browser localStorage
Intuitive, clutter-free interface
Responsive across screens for seamless productivity

# Tech Stack
HTML5 – Semantic layout
CSS3 – Modern UI styling
JavaScript (Vanilla) – Logic & interactivity
LocalStorage – Persistent data retention
